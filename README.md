# snake game in C 🐍
This was a fun little project where I made a simple snake game prototype using C, and only the standard libraries.

What I did:
- Learned ANSI escape sequences to render stuff to the screen with colours, etc.
- Implemented a circular array using a struct.
- Used termios to customize the terminal and to allow for non-blocking & non-buffered controls.
